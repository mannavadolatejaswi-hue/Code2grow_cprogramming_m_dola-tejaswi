Day15-student performance analyzer
The problem asks you to write a C program that takes marks of up to 10 students, calculates their average,
highest, and lowest marks, counts how many passed or failed, and shows the grade distribution based on their marks.
Input: Marks of each student (out of 100)
Output: Average marks, highest and lowest marks, number of passed and failed students
and grade summary (A+, A, B, C, D, F).
